### 复试-面试

一共分为三个面试，综合面试1、综合面试2、英语面试，三者交叉进行。基本上从早一直到晚。  
综合面试1、综合面试2都是3个老师。   
英语面试是5、6个老师，但是只有离你最近的3个老师会问问题。
